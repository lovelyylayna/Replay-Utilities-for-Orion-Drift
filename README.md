# Layna's Replay Utilities script

An orion drift spectator script that allows you to manage replays in bulk!

## Usage

Firstly download the latest [release](https://github.com/lovelyylayna/Replay-Utilities-for-Orion-Drift/releases/latest) and move the script into your Behaviours folder for the spectator camera.
Join an offline server and activate the camera script.

Open camera UI for more actions.
Camera will move like Freecam by default.
